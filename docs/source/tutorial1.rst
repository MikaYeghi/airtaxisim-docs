Tutorial 1: Basic Simulation
============================

This tutorial walks through a minimal AirTaxiSim example.
It is intended as a starting point for new users.

Overview
--------

In this tutorial, you will learn how to:

- Run a basic AirTaxiSim scenario
- Understand the main configuration files
- Inspect simulation outputs

Steps
-----

1. Prepare a configuration file
2. Launch the simulator
3. Observe the results
